Tutorial Bash

Bash es una linea de comando, es la mas usada en linux.

Variables

Las variables son una parte muy importante en cada lenguaje. En ellas podemos guardar informacion para su uso posterior.

Inicializar variables

En bash inicializamos variables con el simbolo =

Por ejemplo, para inicializar la variable "contador" a 0 pondriamos contador=0

Utilizar variables

Podemos usar las variables en bash usando operadores.

Estos son algunos de los operadores más comunes

  • ==
  • !=
  • <
  • >

Carpetas

En bash podemos movernos entre las carpetas, crear carpetas, eliminarlas, etc.

Para movernos a una carpeta usamos cd.

Para crear una carpeta usamos mkdir.

Para borrar una carpeta usamos rmdir pero debe estar vacía.

También podemos mover y copiar carpetas con mv y cp respectivamente.
